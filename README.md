# WendyNevins WordPress Theme

A modern, fresh WordPress theme designed for Wendy Nevins RVN - Registered Veterinary Nurse promoting CPD courses.

## Features

- **Mobile-First Design**: Fully responsive with hamburger menu for mobile
- **Modern CSS**: Uses CSS custom properties (variables), Flexbox, and Grid
- **Scroll Animations**: Smooth fade-in animations as content scrolls into view
- **Accessible**: WCAG compliant with skip links, focus states, and semantic HTML
- **Fast Performance**: Optimized code, no unnecessary scripts
- **Customizable**: Easy to customize with CSS variables

## Brand Colors

The theme maintains the existing brand colors:

- Primary Green: `#0d8f4f`
- Primary Dark: `#0a8043`
- Primary Light: `#66bb6a`
- Primary Lighter: `#e9ffe8`

## Template Files

- `index.php` - Blog post listing
- `single.php` - Single blog post
- `page.php` - Standard page template
- `archive.php` - Category/tag archives
- `search.php` - Search results
- `404.php` - 404 error page
- `header.php` - Site header with navigation
- `footer.php` - Site footer with widgets

## Menu Locations

- **Primary**: Main navigation menu
- **Footer**: Footer links menu

## Widget Areas

- **Sidebar**: Blog sidebar
- **Footer Column 1-4**: Four footer widget columns
